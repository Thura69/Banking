
# Banking online payment web

This is KBZ banking web with blogs and user sign-in/up.


## Features

- Homepage
- Blog
- Single Blog Page
- Login 
- Sign Up



## Tech Stack

React.js, TypeScript, Tailwind, React-router-dom, Git & GitHub
